# IGMPv2Ser

> PyQt and scapy project.

## 介绍

基于PySide6、scapy开发的模拟IGMPv2服务端的发包工具，用于发送IGMPv2协议相关报文。

### Developer

[FanHao](http://alanfanh.github)

### 项目结构

main目录包含源码文件、配置文件
build目录包含界面ui、程序打包文件

## 环境

> python3.10 64bit

### 依赖

> 可使用"pip install -r requirements.txt"一键安装所有依赖项

````text
pyside6=6.2.1
wmi==1.5.1
scapy==2.4.5
````
